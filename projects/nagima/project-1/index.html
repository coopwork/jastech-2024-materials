Мои проекты

Telegram-бот для интернет-магазина

Проект: Телеграм-бот с использованием Fake Store API

Описание: Этот бот предоставляет доступ к спискам продуктов и категорий, используя Fake Store API. Пользователь может выбирать категории и получать информацию о продуктах прямо в Telegram.

AudioTextify

Проект: Веб-приложение для преобразования аудио в текст

Описание: Приложение преобразует аудиофайлы в текст с использованием Python и Flask. В основе проекта лежат библиотеки SpeechRecognition и langid для работы с аудио.

CRM-system

Проект: CRM-система для управления процессами в учебных заведениях

Описание: Система поддерживает работу трех типов пользователей: администратора, преподавателя и ученика, с возможностью управления расписанием, учетными записями.
